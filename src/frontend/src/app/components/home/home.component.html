Leo AI Chat Assistant

Interactive demo of an AI-powered corporate assistant that integrates with Slack, Microsoft Teams, and Telegram. Leo can answer HR questions, troubleshoot IT issues, provide sales analytics, track project status, and retrieve knowledge base information — all powered by Azure OpenAI. Each channel demonstrates a different use case for corporate AI assistants.

How to Use

  1. Select a platform tab — choose between Slack, Microsoft Teams, or Telegram interface.
  2. Browse channels — each channel demonstrates a different AI use case (HR, IT, Sales, Projects, Knowledge Base, Executive).
  3. Tag the bot — type @leo in your message to activate the AI assistant in any channel.
  4. Use the Apps section — in Slack, click "Leo" under Apps for a direct conversation without needing to tag. In Teams and Telegram, select the "Leo" chat.
  5. Ask questions — try HR policy questions, IT troubleshooting, sales analytics, sprint status, or company knowledge queries.
How it works: Each channel uses a different AI system prompt to demonstrate how a single assistant can serve multiple corporate functions. Powered by Azure OpenAI with context-aware responses.
